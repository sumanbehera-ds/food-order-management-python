# 🍔 Food Ordering App (Python CLI)

**Author:** Suman Behera  

A Python CLI food ordering application demonstrating **Object-Oriented Programming (OOP)**, modular design, and interactive user workflows including **menu browsing, cart management, and order processing**.

---

## 📌 About the Project

A simple **Command-Line Food Ordering Application** built using **Python** and **Object-Oriented Programming (OOP)** principles.

This project simulates a basic food ordering system where users can:

- View menu  
- Add items to cart  
- Check current order  
- Place an order  

---

## 🚀 Features

- 📋 View available food items with prices  
- ➕ Add items to your order  
- 🧾 View current order with total price  
- ✅ Place order  
- ❌ Exit anytime
